# Структура реального сайта (с ЛК)

## 1) Публичные страницы
- Главная (продукт, кейсы, CTA)
- B2B‑лендинг (HR‑аналитика)
- FAQ / Условия / Контакты

## 2) Личный кабинет (HR)

### Разделы
- Дашборд (метрики по рынку)
- Отчеты (история запросов, экспорт PDF/CSV)
- Профили ролей (сохраненные роли/навыки)
- Команда (приглашения, роли)
- Настройки (тариф, оплата)

### Ключевые функции
- запрос рынка роли
- конкурентный анализ
- шаблон вакансии
- расписание отчетов

## 3) Админка
- пользователи
- активные подписки
- лимиты запросов
- аудит логов

## 4) Архитектура Web
- Frontend: Next.js
- Backend: Node.js/Express или NestJS
- Auth: email + magic link
- Billing: Stripe / YooKassa
