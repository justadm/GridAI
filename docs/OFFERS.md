# Офферы для разных сегментов HR

## 1) HR‑менеджер (массовый найм)
**Оффер:** быстрый market‑research по роли и городу за 1–2 минуты.  
**Ценность:** меньше времени на сбор рынка, быстрее запуск вакансий.

## 2) HR‑BP / HRD
**Оффер:** агрегированная аналитика рынка и конкурентов.  
**Ценность:** понимание конкурентной позиции и зарплатных вилок.

## 3) Рекрутинговое агентство
**Оффер:** быстрый шаблон вакансии и топ‑навыки по роли.  
**Ценность:** ускорение подготовки вакансий и разговоров с клиентами.

## 4) Tech‑lead / Hiring manager
**Оффер:** требования и тренды рынка по технологии.  
**Ценность:** корректные требования, меньше “перебора” в вакансии.
