# KPI и воронка

## 1) Воронка (B2B)

1. Публикация/контент → просмотр
2. Переход в TG → заявка на пилот
3. Демо (15–20 мин)
4. Пилот (2–3 роли)
5. Подписка

## 2) KPI по этапам

- Просмотры → переходы в TG: **2–5%**
- Переход → заявка на пилот: **5–15%**
- Заявка → демо: **40–60%**
- Демо → пилот: **30–50%**
- Пилот → подписка: **20–40%**

## 3) Метрики на неделю

- число лидов
- число демо
- число пилотов
- cost per lead (CPL)
- conversion rate по этапам

## 4) Цели на 30–60 дней

- 10–20 лидов
- 5–8 демо
- 2–3 пилота
